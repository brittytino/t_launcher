# Contributing

Thank you for your interest in T_Launcher. We welcome contributions that align with our core philosophy of **minimalism and efficiency**.

**Important:** Before opening a PR or Issue, please read our [Contributing Guidelines][contributing] in the root of the repository to understand our strict feature policy.

## Ways to Contribute

*   **Code:** Fix bugs or implement accepted features. Please open an issue first to discuss major changes.
*   **Translations:** Help us reach more users by improving translations on [Weblate][toolate].
    <br><img src=https://toolate.othing.xyz/widget/brittytino-launcher/launcher/horizontal-auto.svg alt=translation status>
*   **Documentation:** Improve guides or clarify the strict philosophy for new users.
