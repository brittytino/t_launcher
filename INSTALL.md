# Installation Guide

## Download T_Launcher

### Option 1: GitHub Releases (Recommended)

1. Visit the [Releases page](https://github.com/brittytino/t_launcher/releases/latest)
2. Download the latest `app-release.apk` or `app-default-release.apk`
3. Follow the installation steps below

### Option 2: F-Droid (Coming Soon)

F-Droid submission is in progress. Once approved, you'll be able to install directly from the F-Droid app.

---

## Installation Steps

### For First-Time Installation

1. **Download the APK**
   - Download from [GitHub Releases](https://github.com/brittytino/t_launcher/releases/latest)

2. **Enable Installation from Unknown Sources**
   - Open **Settings** on your Android device
   - Navigate to **Security** or **Apps & Notifications**
   - Enable **Install from Unknown Sources** for your browser or file manager
   - (On Android 8.0+, you'll be prompted per-app when installing)

3. **Install the APK**
   - Open your **Downloads** folder or notification
   - Tap the downloaded APK file
   - Tap **Install**
   - Wait for installation to complete

4. **Set as Default Launcher**
   - Press the **Home** button
   - Select **T_Launcher** from the list
   - Tap **Always** or **Set as default**

## Updating T_Launcher

### Manual Update

1. Download the new APK from [Releases](https://github.com/brittytino/t_launcher/releases/latest)
2. Install the new APK over the existing installation
3. Your settings and data will be preserved

### Automatic Updates (Coming Soon)

- Once on F-Droid, updates will be automatic
- GitHub users will need to manually check for updates

## System Requirements

- **Minimum:** Android 6.0 (API 23)
- **Target:** Android 15 (API 36)
- **Recommended:** Android 10+ for best performance
- **Storage:** ~20 MB
- **Permissions Required:**
  - None! T_Launcher respects your privacy

---

## Troubleshooting

### Cannot Install APK

**Problem:** "App not installed" error

**Solutions:**
- Ensure you have enough storage space (at least 50 MB free)
- If updating, uninstall the old version first (you'll lose settings)
- Verify the APK downloaded completely (check file size)

### Launcher Not Showing as Option

**Problem:** T_Launcher doesn't appear when pressing Home

**Solutions:**
1. Go to **Settings** ‚Üí **Apps** ‚Üí **Default Apps** ‚Üí **Home App**
2. Select **T_Launcher**
3. If still not visible, restart your device

### App Crashes on Startup

**Problem:** App closes immediately after opening

**Solutions:**
- Clear app data: **Settings** ‚Üí **Apps** ‚Üí **T_Launcher** ‚Üí **Storage** ‚Üí **Clear Data**
- Reinstall the app
- Report the issue on [GitHub Issues](https://github.com/brittytino/t_launcher/issues)

---

## Uninstallation

1. Switch to your previous launcher:
   - Go to **Settings** ‚Üí **Apps** ‚Üí **Default Apps** ‚Üí **Home App**
   - Select a different launcher
2. Uninstall T_Launcher:
   - Go to **Settings** ‚Üí **Apps** ‚Üí **T_Launcher**
   - Tap **Uninstall**
